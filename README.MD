Simple vue authentication with frontend and backend

<h3>
For frontend we use

vue 3 composition api and vuex

For backend we use

express

mongodb

bcryptjs

morgan and

jsonwebtoken

</h3>

To use https://github.com/skarthickkannan/vue-boilerplate.git

To run and install frontend

```js
cd client
```

```js
npm install
```

```js
npm run serve
```

To run and install backend

```js
cd server
```

```js
npm install
```

```js
npm start
```
